sinatra-ToDoMVC
===============

ToDo application using Sinatra, DataMapper, &amp; MySQL


apt repos:
---------

sudo apt-get install -y rubygems libmysqlclient-dev

(libsqlite3-dev)

gems:
----

sudo gem install sinatra sinatra-contrib data_mapper dm-mysql-adapter --no-rdoc --no-ri

(dm-sqlite-adapter)

running:
--------

cd sinatra-ToDoMVC
ruby app.rb
